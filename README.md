# Multi-Tape Turing Machine Simulator

## Обзор

Интерактивный веб-симулятор **многоленточной машины Тьюринга**, реализованный на чистом JavaScript (без фреймворков).

Проект позволяет:
- моделировать машины Тьюринга с несколькими лентами,
- создавать и редактировать собственные программы,
- запускать симуляцию пошагово или автоматически,
- визуально отслеживать процесс вычислений,
- изучать теоретическую эквивалентность многоленточных и одноленточных машин.

Проект ориентирован на обучение и демонстрацию фундаментальных концепций теории вычислений.

## Эквивалентность машин Тьюринга

Проект демонстрирует, что:

> любая многоленточная машина Тьюринга может быть преобразована в одноленточную.

Реализовано:

* объединение лент через разделители
* маркировка головок
* визуальная демонстрация

 При этом сложность возрастает до **O(T²)**
